# Brogress Bars Backgrounds Whit Aanimations CSS

Add Aanimations CSS to your progress bars by add class `.animated .fadeInLeft`

```html
    <div class="progress mb-1">
        <div class="progress-bar bg-success animated fadeInLeft" role="progressbar" style="width: 25%" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100">
    </div>
    </div>
    <div class="progress mb-1">
        <div class="progress-bar bg-primary animated fadeInLeft" role="progressbar" style="width: 50%" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
    </div>
    <div class="progress mb-1">
        <div class="progress-bar bg-warning animated fadeInLeft" role="progressbar" style="width: 75%" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
    </div>
    <div class="progress mb-1">
        <div class="progress-bar bg-danger animated fadeInLeft" role="progressbar" style="width: 100%" aria-valuenow="100" aria-valuemin="0" aria-valuemax="100"></div>
    </div>
```
